Estratégias de IA: Classificação e Modelagem 🤖

Este repositório contém o desenvolvimento de um modelo de Inteligência Artificial para classificação de dados, focado em estratégias de aprendizado de máquina.

🚀 Funcionalidades
- Pré-processamento e limpeza de dados (Data Cleaning).
- Treinamento de modelos de classificação.
- Avaliação de métricas de performance (Acurácia, Precisão, Recall).

🛠️ Tecnologias
- Python (Pandas, Scikit-Learn, NumPy)
- Google Colab para ambiente de desenvolvimento.

Projeto desenvolvido para a disciplina de Inteligencia Artificial na UFOP.
